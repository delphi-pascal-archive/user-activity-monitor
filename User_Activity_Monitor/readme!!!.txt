Программа предназначена для регистрации действий пользователей, работающих за одним компьютером.

Возможности программы:
- регистрация действий неограниченного числа пользователей работающих за одним компьютером; 
- ведение лога времени захода на компьютер конкретного пользователя и выхода из него;
- ведение лога нажатых пользователями клавиш;
- ведение лога посещенных пользователями сайтов;
- учет времени, в которое были нажаты какие-либо клавиши, либо был осуществлен заход
  на какой-либо сайт;
- автоматическое завершение сеанса пользователя, если пользователь бездействовал более N
  минут;
- автоматическое выключение компьютера в заданное время.

Пограмма автоматически запускается при запуске Windows (при первом запуске программа
записывается в секцию автозагрузки) поэтому СРАЗУ определитесь с местоположением программы
на диске.

// Важно!

При первом запуске программы, для входа в нее введите нижестоящие логин и пароль:
 Логин: Admin
 Пароль: Admin.

Затем ОБЯЗАТЕЛЬНО измените пароль Администратора (логин оставьте Admin, так как только под
"Админом" разрешено вызывать Диспетчер задач и заходить в реестр. Для остальных пользователей
это запрещено).

Изменять время простоя можно только в режиме Администратора.

Логины и пароли пользователей изменяйте/добавляйте по усмотрению.
